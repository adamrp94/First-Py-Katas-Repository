# First-Py-Katas-Repository

#(Katas de Lógica)

En este repositorio he ido subiendo las soluciones a los ejercicios de lógica que hemos trabajado. El objetivo ha sido practicar desde lo más básico de Python hasta cosas un poco más avanzadas como clases y objetos.

## ¿Qué hay en este proyecto?

He metido todos los ejercicios en un solo archivo llamado `Katas_first_exercise.py`. Para que sea fácil de revisar, he puesto el enunciado de cada ejercicio justo encima del código como un comentario. 

También he añadido algunos `print` al final de cada función comoejemplo.

## Temas que he practicado:
* **Funciones y lógica:** Crear funciones para procesar textos, números y listas.
* **Uso de Map, Filter y Reduce:** Para trabajar con listas de forma más eficiente.
* **Clases (POO):** He creado un par de clases (`Arbol` y `UsuarioBanco`) para entender cómo funcionan los objetos y sus métodos.
* **Manejo de errores:** He usado `try/except` para que el programa no se rompa si el usuario mete un dato que no toca.



## Pasos que he seguido:
1. Primero configuré el repositorio aquí en GitHub para tenerlo todo organizado.
2. Fui resolviendo los ejercicios uno a uno, probando que funcionaran.
3. Al final, limpié un poco el código para que los nombres de las funciones y los comentarios se entiendan bien.

Espero que el código esté claro y se ajuste a lo que pedía el ejercicio.
